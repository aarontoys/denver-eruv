Basic App Usage
  • Start with the Map Tab. The Map Tab should automatically determine you geolocation (assuming you allow locations services).
  • Tap on the map to add a pin. This action records the latitude and longitude of the pin.
  • After placing a pin, the app will auto-navigate you to the Report Form Tab and load the nearest address to the pin.
  • Next, complete the report by selecting issue & severity and adding comments and selecting whether or not a bucket truck is required.
  • Next, click the camera icon to take a photo. If the photo is too large, you should receive a message asking you to retake the photo. Please note this feature only works on mobile devices
  • If the photo is not too large (and if there is a photo), you can then submit your report
  • All reported items can be found on the Report View tab. These are grouped by severity and the truck icon indicates that a bucket truck is needed for the fix.
  • You can click on any item for more detais. This will show a larger picture and a map of where the problem is. Please DO NOT click the green checkbox as this will mark the item as completed/fixed.
  • The Sections Tab is not yet complete. At a later stage, this will help with ensuring that each section is covered by someone and provide statuses of each section, i.e. up/down, etc.
